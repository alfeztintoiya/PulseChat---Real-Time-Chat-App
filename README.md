## ğŸš€ Getting Started

### Prerequisites

- Node.js 16+ and npm
- Modern web browser

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/alfeztintoiya/pulsechat.git
   cd pulsechat
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run tests
